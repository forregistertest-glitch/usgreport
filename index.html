Technical Documentation
==============================================================================
PROJECT: USG Reporting System (LAN-Based)
==============================================================================

TECHNOLOGY STACK
------------------------------------------------------------------------------
- HTML5 / CSS3 / JavaScript (ES6+)
- Local Fonts (Sarabun .ttf)
- URLSearchParams for cross-page data transfer
- Base64 encoding for cross-machine sharing

KEY TECHNIQUES
------------------------------------------------------------------------------
1. Table Wrapper Technique for Print Layout
   - Fixed header/footer without content overlap
   
2. Cross-Machine Data Transfer
   - Base64 encoded JSON via URL parameters
   - LocalStorage as fallback

SERVER COMMANDS (LAN Sharing)
------------------------------------------------------------------------------
Start:  npx -y http-server -p 8080 -a 0.0.0.0 -c-1
Stop:   Ctrl + C

URL:    http://[YOUR_IP]:8080/index.html
